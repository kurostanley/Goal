# Goals

Goals is a web API that helps users reach their goals by visualizing their progress. The app was developed using Node.js, Express.js, MySQL, Passport.js and Express-session. The app was designed to be user-friendly and easy to use, with a modern interface that makes it easy for users to track their progress and stay motivated.

## Features
- A goal tracking system that allows users to set, track, and visualize their progress towards their goals.
- User authentication and authorization using Passport.js and Express-session, so users can securely create and manage their accounts.
- A REST API that allows users to easily interact with the app, and detailed documentation for the team.
- CRUD API endpoints that allow users to create, read, update, and delete their goals.
- AWS RDS cloud server for easy access and scalability.
- Solved cross-origin cookies sharing issues, ensuring seamless integration with the front-end.

## Usage

To use the app, simply sign up for an account and start creating your goals. You can track your progress and visualize your progress, so you can stay motivated and reach your goals.
